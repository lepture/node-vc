# vc

> Change the version of a package.json.

-----

## Install

Install vc with npm:

    $ npm install vc -g

## Usage

It is pretty easy.

Get the help menu:

    $ vc -h

Show current version:

    $ vc

Set a new version:

    $ vc 1.0.0

Increase the patch number by 1:

    $ vc -c
